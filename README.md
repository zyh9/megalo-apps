### 静态资源拷贝

```javascript
    chainConfig.plugin('copy-webpack-plugin')
      .use(
        CopyWebpackPlugin,
        [
          [{
            from: resolve(__dirname, './src/static'),
            to: resolve(`dist-${process.env.PLATFORM}/static`)
          }]
        ]
      )
```
